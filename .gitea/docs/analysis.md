# Full Architectural Design Documentation
## Περιγραφή

Το go-reloaded είναι ένα πρόγραμμα που μορφοποιεί και διορθώνει ένα κείμενο για τον αναγνώστη, εφαρμόζοντας κανόνες μορφοποίησης και αυτοδιόρθωσης. Το πρόγραμμα έχει modular σχεδίαση (pipeline) που διευκολύνει την εφαρμογή κανόνων και την επεκτασιμότητα.

## Pipeline Overview

Το πρόγραμμα χωρίζεται σε τρία βασικά στάδια:

Input Stage – Ανάγνωση αρχείου

Processing Stage – Εφαρμογή κανόνων (tokenization, rule processing, formatting)

Output Stage – Εξαγωγή του επεξεργασμένου κειμένου σε νέο αρχείο

## Stages & Modules
### Input Reader

Διαβάζει το κείμενο από ένα ξεχωριστό αρχείο.

Αποθηκεύει το περιεχόμενο ως string.

Επιστρέφει το string στο επόμενο στάδιο.

### Tokenizer

Χωρίζει το string σε tokens, όπου κάθε token είναι μία λέξη ή ένα σημείο στίξης.

Τα σημεία στίξης δεν μετρούν ως λέξεις, αλλά διατηρούνται ξεχωριστά.

Αναγνωρίζει παρενθέσεις ως ειδικές οδηγίες (rules) που θα επηρεάσουν τις προηγούμενες λέξεις.

### Rule Processor

Ελέγχει κάθε token και εφαρμόζει τους κανόνες μορφοποίησης:

Κανόνες για παρενθέσεις: εφαρμόζονται στην προηγούμενη λέξη.

Κανόνες για σημεία στίξης: εφαρμόζονται κάθε φορά πριν ή μετά το σημείο στίξης.

Κανόνες για άρθρα: μετατρέπει a σε an αν η επόμενη λέξη ξεκινάει με φωνήεν.

Κρατάει λίστα με τα tokens που άλλαξαν ώστε να μην γίνονται διπλές αλλαγές.

Επιστρέφει επεξεργασμένα tokens στο επόμενο στάδιο.

### Formatter / Output Writer

Λαμβάνει τα επεξεργασμένα tokens και συνθέτει το τελικό κείμενο.

Κρατάει τη σωστή σειρά των tokens.

Εφαρμόζει formatting για τα σημεία στίξης.

Αποθηκεύει το τελικό κείμενο σε ένα νέο αρχείο.

## Ροή Δεδομένων
Input File -> Input Reader -> Tokenizer -> Rule Processor -> Formatter -> Output File
Κάθε στάδιο παίρνει ως input το αποτέλεσμα του προηγούμενου.

Το pipeline επιτρέπει εύκολη προσθήκη νέων κανόνων στο μέλλον χωρίς να αλλάξει η συνολική δομή.

## Modules / Functions (High-Level)

main.go: Εκκινεί το πρόγραμμα, διαβάζει arguments (input/output).

input_reader.go: Ανάγνωση αρχείου.

tokenizer.go: Διαχωρισμός σε tokens.

rule_processor.go: Εφαρμογή κανόνων.

formatter.go: Σύνθεση τελικού κειμένου.

utils.go (προαιρετικό): Βοηθητικές συναρτήσεις για capitalization, ελέγχους κλπ.
