# go-reloaded

Το go-reloaded είναι ένα πρόγραμμα που διορθώνει ένα κείμενο για τον αναγνώστη, εφαρμόζοντας κανόνες μορφοποίησης και αυτοδιόρθωσης.

## Installation & Usage
1.Κλωνοποιήστε το repository:
```bash
git clone https://platform.zone01.gr/git/vxanthio/go-reloaded.git
```  
2.Μεταβείτε στον φάκελο του project:
```bash
cd go-reloaded
```
3.Τρέξτε το πρόγραμμα με το Go:
```bash
go run main.go <input_file> <output_file>
```
